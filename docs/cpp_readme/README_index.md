# KidMagic 源码阅读导航

这个目录用于帮助初学者按模块阅读 C++ 代码。

推荐阅读顺序：
1. README_main.md（先看总调度）
2. README_module_button.md（输入）
3. README_module_led.md（灯光输出）
4. README_module_ui.md（OLED 与蜂鸣器）
5. README_module_audio.md（I2S 录音）
6. README_module_network.md（WiFi 与 HTTP）

源码文件对应关系：
1. src/main.cpp -> README_main.md
2. src/module_button.cpp + src/module_button.h -> README_module_button.md
3. src/module_led.cpp + src/module_led.h -> README_module_led.md
4. src/module_ui.cpp + src/module_ui.h -> README_module_ui.md
5. src/module_audio.cpp + src/module_audio.h -> README_module_audio.md
6. src/module_network.cpp + src/module_network.h -> README_module_network.md
